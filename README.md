# 🎬 CineSearch | Movie Finder App

A sleek, modern movie search engine built with Vanilla JavaScript, CSS3, and HTML5. 

## ✨ Features
* **Real-time Search:** Fetches movie data dynamically from the OMDb API.
* **Theme Toggle:** Smooth transition between Dark and Light modes.
* **Persistence:** Uses `localStorage` to remember your theme preference.
* **Responsive Grid:** A flexible layout that looks great on desktop and mobile.
* **Smart Placeholders:** Automatically handles missing movie posters with a custom "No Poster" image.

## 🛠️ Built With
* **JavaScript (ES6+):** `Async/Await` for API calls and `Array.map()` for DOM injection.
* **CSS3:** Custom properties (variables) for the theme engine and CSS Grid for the layout.
* **HTML5:** Semantic structure for better accessibility.
* **API:** [OMDb API](http://www.omdbapi.com/)
